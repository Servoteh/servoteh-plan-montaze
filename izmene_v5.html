Promena modela: project → work_packages → phases

A) Šta je promenjeno u modelu

Stari model (v4.x)Novi model (v5.0)
MACHINES[] hardkodiran nizprojects[] → workPackages[] → phases[] dinamički
allState[machineId] = phases[]allData.projects[].workPackages[].phases[]
Machine tabs (fiksni)Project selector + WP tabs (dinamički)
project_machine_meta odvojenoMeta integrisano u projects objekat
Jedan implicitni projekatViše projekata istovremeno
localStorage key per machineJedan key sa celom strukturom

Novi UI tok

  • Project selector na vrhu (dropdown + dugmad za dodavanje)
  • Work package tabs ispod projekta
  • Plan/Gantt/Total prikazuju faze izabranog WP-a
  • Meta podaci (RN, PM, rok, email) u popup editoru po projektu
  • Dashboard prikazuje projekat + trenutnu poziciju

Migracija sa v4

Automatska migracija: ako postoji plan_kovacka_linija_v4 u localStorage, stare mašine se pretvaraju u work_packages jednog projekta.
